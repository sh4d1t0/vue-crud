# vue-users-crud

Example to vue user system with taildwind, typescript and pinia

1.  **Clone repo.**

    ```shell
    git clone https://github.com/sh4d1t0/vue-crud.git
    ```

2.  **Install.**

    ```shell
    cd vue-crud/
    npm install
    ```

3.  **Run.**

    ```shell
    npm run dev
    ```
